# My Budget Tracker

<img src=images/1.png alt="Screenshot" width="90%" height="90%" />

**My Budget Tracker** 은 C#에서 MySQL DB를 연동하여 상호 작용하는 <u>연습용 기입장 GUI 애플리케이션</u>입니다.

+ 지정된 사용자와 비밀번호로 localhost에 로그인합니다.
+ 수입, 지출, 금액, 내역 등을 입력한 후 DB와 연동될 수 있습니다.
+ C# WPF .Net FrameWork를 사용해서 GUI를 Windows Application으로 구성하였습니다.
